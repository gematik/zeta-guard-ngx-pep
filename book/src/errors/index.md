# Error Reference

This site contains “read more” pages linked from error responses generated from the PEP
(`error_uri`).
- If you got here from an error: open the linked page for details.
- If you’re looking for a specific error, use the *search* (top left) or open **Errors**.
- If the error you received is not listed here, it was most likely generated by an
  upstream backend and just passed through PEP. Consult its documentation.
